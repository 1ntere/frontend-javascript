    해당 태그에 있는 클래스 중 아직 안 배운 내용(integrity, crossorigin)

    integrity 란?
        진실되다, 링크를 안전하게 가져오기 위해 설정하는 속성 (필수는 아니지만, 보안, 신뢰성을 위해 사용하는 것이 좋음)

    crossorigin 이란?
        annonymous : 자격 증명을 포함하지 않도록 설정 (필수는 아니지만, 보안, 신뢰성을 위해 사용하는 것이 좋음)

라이브러리와 프레임워크 차이

    라이브러리 vs 프레임워크

    라이브러리 = 샤브샤브뷔페 느낌
        재료가 다 갖추어져 있어서 내가 요리를 다양하게 할 수 있음
        개발자가 개발을 하는데 있어 활용범위가 더 넓음

    프레임워크 = 샤브샤브밀키트 느낌
        재료의 양이 한정되어 있어서 내가 할 수 있는 요리가 제한적임
        개발자가 개발을 하는데 있어 활용범위가 제한적임
        ex. 부트스트랩은 프레임워크이기 때문에 어느정도 기본 스타일은 제공해주지만
            그 외의 추가적인 스타일이 필요할 경우 css를 사용해서 디자인을 만들어줘야 함

class 사용 방법 정리

    container  : 반응형 고정 폭을 제공하는 역할
                 페이지 콘텐츠를 정렬하고 중앙에 배치
                 기본적으로 페이지의 양 옆에 여백을 줘서 콘텐츠를 중앙에 정렬

    row        : container에서 가로 1줄을 나타냄
                 가로 1줄을 col-12(세로 12칸)로 이루어져 있음
    col-n      : row(가로 1줄)에서 세로로 몇 칸(n 칸)을 차지할 것인지 나타냄
                 col-n이 12를 초과하는 경우 다음 줄로 자동으로 이동함
        col-1    : 세로로 중간사이즈로 1칸
        col-xs-1 : 세로로 가장 작은 사이즈로 1칸 차지
        col-4    : 가로 1줄에서 4칸을 차지하게 되며
                   가로 1줄은 총 12칸이기 때문에 여유 칸은 8칸이 됨

    card       : 부트스트랩에 있는 카드 스타일을 적용
        card-body : 카드 본문 틀, 틀 안에 img, title, text 등을 넣어줌
            card-title   : 카드 본문에서 제목을 나타냄
                           제목 스타일을 적용
            card-text    : 카드에서 내용을 작성하는 공간
                           텍스트 스타일을 적용
            card-img-위치 : 카드에 있는 img의 위치를 적용
                            img가 카드 바깥으로 나가는 것을 방지
                            카드 틀 안에 img가 있을 수 있도록 도와줌
                            width와 height를 사용하지 않아도 이미지의 크기가 자동으로 조정됨
                card-img-top    : 이미지를 카드 본문 중   위에 배치
                card-img-bottom : 이미지를 카드 본문 중 아래에 배치

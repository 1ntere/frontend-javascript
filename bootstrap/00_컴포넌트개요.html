    독립적으로 설계된 기능, 재사용이 가능한 모듈 또는 단위
    객체, 기능, 메서드 등

Button태그

    클릭 가능한 버튼을 생성
    부트스트랩으로 버튼을 시작할 때
    class 내부에 btn 버튼 시작 기능을 작성해주고
    추가로 작성하고 싶은 스타일을 적용

    btn-primary : 파란색

버튼 테두리 표시하기

버튼 크기 조정

    버튼의 크기는 btn-lg 나 btn-sm 와 같은 크기 조정을 사용해서 적용
    padding을 이용해서 버튼의 크기를 조정할 수 있음

버튼 비활성화

    버튼을 비활성화 하기 위해서는 disabled를 추가하면 됨

    하지만 a태그 버튼을 비활성화 하기 위해서는
    disabled와 aria-disabled="true"를 같이 사용해주어야 함

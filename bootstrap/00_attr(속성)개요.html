attr

    attr (attribute) 속성
    jQuery에서 HTML 태그 속성을 가져오거나 설정할 때 사용하는 기능

    예제
    
    img 태그에서 src, alt, style, class 모두 sttr에 해당 함
    한 마디로, attr은 태그 안에 작성된 속성값을 가져오거나 변경하거나 제거할 때 사용

    1. 태그 안에 있는 속성값 가져오기
        $("img").attr('src');
            //src안에 있는 그림 주소 값을 가져오는 것
    2. 태그 안에 있는 속성값 변경하기
        $("img").attr('src', '변경할주소값설정');
            //src안에 있는 그림 주소 값을 변경하는 것(이미지 변경)
    3. 태그 안에 있는 속성값을 여러 개 변경하기
        $("img").attr({src: '변경할주소값설정', alt: "변경된이미지설명"});
            //src와 alt를 한 번에 변경해줌

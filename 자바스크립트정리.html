JavaScript 개념 정리

1. var, let, const 차이점에 대해 작성하시오

    var : 재선언 가능, 재할당(값 변경) 가능
          사용을 거의 하지 않음
        ex. var 변수 = 0;
            var 변수 = 1;//재선언 가능
                변수 = 2;//재할당 가능
    let : 재선언 불가능, 재할당(값 변경) 가능
          java에서 변수와 같은 존재
        ex. let 변수 = 0;
            let 변수 = 1;//재선언 불가능
                변수 = 2;//재할당 가능
    const : 재선언 불가능, 재할당(값 변경) 불가능
            java에서 상수와 같은 존재
        ex. const 변수 = 0;
                  변수 = 1;//재선언 불가능
                  변수 = 2;//재할당 불가능
    
    재선언과 재할당이란?
    재선언 : 다시 선언하는것
                 선언 : 자신의 생각이나 입장을 분명하게 하는 것
             다시 변수명에 대해서 정의를 내리는 것
    재할당 : 값에 새로운 값을 저장하는 것(값 변경)
자료형 재선언 재할당
var O O
let X O
const X X

3. JavaScript에서 객체에 대해 작성하시오

    객체는 자바스크립트에서 데이터를 구조화하여 저장하는데 사용하는 기본 단위
    키(key)-값(value) 쌍으로 이루어져 있음
    객체는 {} 중괄호를 사용해서 정의
    키는 문자열로, 값은 어떠한 데이터 타입도 가능
    
    예시
    const 사람객체 = {
        '이름' : '홍길동',
        '나이' : 30,
        '인사말' : function() {
            alert('환영합니다.');
        }
    }

    사람 객체는 '이름'과 '나이'라는 속성을 가지고, '인사말'이라는 메서드를 가지고 있음

    사람객체
        속성     : 이름, 나이
        메서드   : 인사말

4. DOM에 대해 작성하시오

    DOM (Document Object Model)
    HTML 문서를 구조화된 표현으로 나타냄
    문서의 각 태그를 객체로 표현한 것

    
    버튼 태그를 객체로 표현한다면?
    const button = {
        type : "button",
        class : 'btn',
        클릭버튼() : function() {
            alert("버튼이 클릭되었습니다.");
        }
    }
    button 객체
        속성        : type, class
        메서드(기능) : 클릭버튼()

5. 백엔드 / 프론트엔드 / 풀스택 개발자의 차이점에 대해 작성하시오.

    백엔드
        서버, 데이터베이스, 데이터베이스와 프론트엔드를 연결해주는 역할
        서버 언어(java, javascript(node.js), python, ...)
        데이터베이스 언어(SQL(oracle, mysqp, ...))
        주소 값을 통한 개발 및 서버관리, 데이터 저장 및 처리

    프론트엔드
        HTML, CSS, JavaScript 등을 사용해서 웹페이지를 구성하고 디자인
        쿠키, 세션, 캐시를 관리하기도 함
        라이브러리나 프레임워크 (Bootstrap, React, jQuery, Vue, ...)등을 사용해서 개발

    풀스택
        프론트엔드와 백엔드 개발을 모두 수행함
        전체적인 구조를 이해하고, 클라이언트와 서버 측 모두에서 작업 가능

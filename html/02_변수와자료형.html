변수 (Variable)

    변하는 수
    메모리에 값을 계속 바꿔가면서 저장할 수 있는 공간

    변수에 이름(변수명)을 붙이고 원하는 값을 넣어주면
    변수명이 필요한 곳에서 변수명을 호출해서 지정된 값을 사용할 수 있음

    
변수명 작명 규칙

    의미 있는 이름 작성(a, b, c 이런 식으로 작성하지 말 것)
    대, 소문자 구별해서 작성
    특수문자는 _(언더스코어, 언더바) 만 사용가능 (단, 첫 글자에는 작성 불가)
    카멜표기법 - 중간에 대문자를 작성해주는 표기방식

    [카멜 표기법(Camel Case)]
    Camel : 낙타
    문장에서 여러 단어를 변수명으로 작성할 때 이어지는 첫 번째 단어를 대문자로 작성
        member name => memberName
    
    특수문자 중에서 $표시가 있음
        $는 혼자 사용할 수도 있고
        $숫자 처럼 사용가능
        $변수명 처럼 맨 앞에도 사용 가능
            추후 jQuery 표기 문법으로 사용할 문자
        $ 는 표현식을 나타낼 때 사용하기도 함

JavaScript 변수의 종류와 특징

    Java에서 사용하던 int, char, String 같은 자료형 x
    
    var     : 변수, var의 변수명 중복 가능(덮어쓰기 가능)
            => 변수의 값 변경 가능, 변수명 재사용 가능
    let     : 변수, let의 변수명 중복 불가능
             => 변수의 값 변경 가능, 변수명 재사용 불가능
    const   : 상수, const의 변수명 중복할 수 없음
            가장 많이 사용
            => 변수의 값 변경 불가능, 변수명 재사용 불가능

자바스크립트의 자료형

    문자열을 String 사용 대신에 "" 를 사용하여 작성
        const 문자열 = "hihi";
    정수는 int 사용 대신에 숫자 작성
        const 정수 = 1;
    논리값은 boolean 대신에 true 또는 false 작성
        const 논리값 = true; //진실
    null : 참조하는 값이 없음, 주로 오타에서 발생

요소 접근 방법

DOM(Document Object Model)

    웹 문서(html)의 모든 태그를 객체 형식으로 표현한 것
    (html 태그(body, h1, h2,,, pre, span 등)을 javaScript에서는 객체라고 표현)

    ex. body 태그 = 자바스크립트에서 body 객체

    html 문서 내에서 특정 태그에 접근 가능
    document.getElementById("id값");을 이용해서
        특정 태그에 접근해서 속성 값을 새로 만들거나 변경하거나 가져올 수 있음
        id값으로 지정한 태그의 정보를 가져온다는 의미

DOM을 이용한 태그 접근하기

    1. id로 접근하기 : document.getElementById("id값");
        id가 일치하는 태그를 가져옴
    2. class로 접근하기 : document.getElementByClassName("class값");
        class가 일치하는 태그를 가져옴
        class가 일치하는 태그가 많을 경우(1개가 아닌 경우)
            모두 가져와서 배열 형태로 저장한다음 전달
    3. name으로 접근하기 : document.getElementByName("name값");
        name이 일치하는 태그를 가져옴

태그 속성 정리

    DOM을 이용해서 태그를 접근하고, 접근한 태그를 추가, 수정, 제거 해주기

    textContent : 태그에서 자식 태그의 모든 텍스트 내용을 보여줌
                html 태그를 무시하고 텍스트 만을 가져오거나 설정할 때 사용
                스타일 변경이 쉽지 않음
                외부 공격을 방지할 때 사용
                innerText와 같지만 다른 방식으로 작동
                
    innerText : 태그에서 자식 태그의 모든 텍스트 내용을 보여줌
                html 태그를 무시하고 텍스트 만을 가져오거나 설정할 때 사용
                css를 새로 설정해서 적용하거나 추가적인 스타일 변경이 필요한 경우 사용
                외부 공격 방지하지 못함
    
    value : input이나 textarea 등의 값을 가져오거나 설정할 때 사용
            이외 체크박스, 라디오 버튼 등 다양한 폼 태그 값을 다룰 때 유용

    innerHTML : html 안에 html 만들어서 작성

    style : html 태그의 스타일을 설정하거나 가져올 때 사용
            스타일을 유동적으로 변경할 수 있음
            색상, 글꼴, 배경색 등을 설정할 수 있음
            보이기, 숨기기도 가능

    className : html 태그에서 클래스 이름을 가져오거나 설정할 때 사용

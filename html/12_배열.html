배열 Array

    변수를 묶음으로 다루는 것

    자료형 상관 없이 여러 값을 한번에 다룸

    JavaScript 배열 특징

    1. 자료형 제한 없음 (String, int 구분하지 않음)
        하지만 보통은 1가지 자료형만 저장
    2. 길이 제한 없음 (자동으로 길이가 늘었다, 줄었다 함)

    배열 선언 방법
    1. const arr = new Array();//0칸짜리 비어있는 배열 생성
    2. const arr = new Array(3);//3칸짜리 비어있는 배열 생성
    3. const arr = [];//0칸짜리 비어있는 배열 생성, 1번과 똑같음
    4. const arr = [값1, 값2, 값3];//3칸짜리 배열 생성, 각 칸의 값이 채워져 있음

    자바에서는 Array와 ArrayList 등등 모두 구분이 되었는데
    자바스크립트에서는 따로 구분을 안하고 자바의 ArrayList와 비슷하다.

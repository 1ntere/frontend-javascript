조건문

    프로그래밍 코드가 
    작성된 조건식에

if문

            
    if(조건식) {
        조건식이 true일 경우 실행할 코드;
    }
            
        조건식 : 결과가 true 또는 false인 식

if-else문

            
    if(조건식) {
        조건식이 true일 경우 실행할 코드;
    } else {
        조건식이 false일 경우 실행할 코드;
    }
            
        else는 조건식을 작성하는 ()가 없음
0~100 사이에 랜덤 숫자를 발생시켜서 홀/짝 판별
